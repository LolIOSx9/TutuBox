ffapple proxy (Wifi Network)

1. Open your Settings

2. Select your "Wifi network -> (i) -> Configure Proxy -> Automatic"

3. Enter at the "URL" section "http://ffapple.com"

4. Disconnect from your Wifi Network and re-connect

5. Clear Safari History cache

You should now be able to download tutubox :).

DNSCloak (4G/Cellulra Data)

1. Download DNSCloak and disable the ffapple proxy

2. Download this Text File and save it to your iPhone

3. Open DNSCloak.

4. Tab on the 3 lines in the left corner

5. In DNSCloak Settings turn on "Connect on Demand"

6. Scroll in DNSCloak settings a bit down and tab on "Blacklist and Whitelist"..

7. Then turn on "Enable Blacklist", a file menu will open and select the earlier downloaded file. (if you already enabled this options just tab on "Pick Blacklist File..")

8. Go back to the main page and Select the DNSServer "a-and-a".

9. Now open settings. Got to and turn on "General -> VPN -> DNSCloak -> (i) -> Connect on Demand"

10. Clear Safari History Cache.

You should now be able to download tutubox :).
